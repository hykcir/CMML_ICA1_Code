# CMML_ICA1_Code

All R codes used for model building (including refinements for original codes) and data analysis and as follow:

1. Models_comparison.R: Compares the performace of the three models under both fixed and variable platform condition

2. Combined_ratio_tunning.R: Compares the performance of the combined model under different weight ratio between place-cell and distance-cell (and under fixed and variable platform condition)

3. Parameters_tunning.R: Compares the performance of the three models under different key parameters (e.g. alpha, gamma, beta) (and under fixed and variable platform condition)

P.S. Fixed parameters and run_trials functions used in Combined_ratio_tunning.R and Parameters_tunning.R are inherited from the default settings in Models_comparison.R
